# Font installation
Download `Hack Nerd Fonts` from www.nerdfonts.com and put in `/usr/share/fonts`

Eventually just get `nerds-font-hack` AUR package

Make sure the font is installed with:
```
> fc-cache --really-force
> fc-list | grep "Nerd"
```
